# **App Name**: KoreAuth

## Core Features:

- Identity-CRM Sync: Orchestrates Next.js 15 sessions with GoHighLevel V2 searchContacts and upsertContact services for unified enterprise data.
- AI Prospect Insights: A generative tool that analyzes appointment history to produce synthesized executive summaries and recommended contact notes automatically.
- Adaptive JWT Shield: Secure middleware implementation managing JWT verification, session persistence, and HTTP-only cookie rotation.
- Multi-Step Auth Flow: Highly responsive, multi-step signup experience featuring real-time Zod validation and interactive password strength feedback.
- LeadConnector Intelligence Hub: Unified dashboard surface that visualizes profile editing, note history, and real-time appointment tracking via API interceptors.
- Security Surface UI: Role-based interface controls including rate-limit feedback, brute-force UI handlers, and masked password toggles built with shadcn/ui.

## Style Guidelines:

- Primary color: Vibrant Indigo (#4B46E5) evoking precision and enterprise reliability.
- Background color: Deep Obsidian (#121214) creating a premium dark-mode foundation similar to Vercel.
- Accent color: Intelligent Azure (#2E90FA) for call-to-action buttons and interactive states, offering a professional contrast.
- Font pairing: 'Space Grotesk' (sans-serif) for technical headlines and 'Inter' (sans-serif) for crisp, machine-objective body text.
- Modern sidebar navigation layout with persistent glassmorphism overlays and fixed-width dashboard containers for desktop precision.
- Low-friction motion design using Framer Motion for sequential auth-form reveals and subtle skeleton pulsing for asynchronous CRM data loading.
- Thin-stroke, minimal line icons provided by Lucide-React to maintain the high-end Stripe and Linear aesthetic.
