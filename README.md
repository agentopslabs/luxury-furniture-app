# KoreAuth | Enterprise Identity Hub

Secure authentication and CRM synchronization platform built with Next.js, ShadCN UI, and Genkit.

## Getting Started

To run the development server (available at http://localhost:9002):

```bash
npm run dev
```

To run the Genkit Developer UI:

```bash
npm run genkit:dev
```

## Features

- **Dashboard**: Intelligence Hub with live GHL V2 sync.
- **Conversations**: Live multi-channel messaging (SMS/Email).
- **Marketing**: Full Social Planner suite and campaign tracking.
- **Pipeline & Opportunities**: Kanban-style deal flow management.
- **Payments**: Order, Invoice, and Transaction synchronization.
- **Contacts**: Real-time contact record management.
- **AI Intelligence**: Automated executive summaries for contacts using Genkit.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS & ShadCN UI
- **AI Engine**: Genkit with Google Gemini
- **Backend Sync**: LeadConnector (GHL) V2 API
